# react-native-hiragana
A React Native module for converting Japanese sentence to hiragana
## Installation

```sh
npm install react-native-hiragana
```

## Usage

```js
import { multiply } from "react-native-hiragana";

// ...

const result = await multiply(3, 7);
```
